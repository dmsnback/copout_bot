### Запуск
#
### Локальный запуск

 - Клонировать репозиторий.
```shell
git clone git@github.com:dmsnback/copout_bot.git
```
- Перейти в директорию.
```shell
cd copout_bot
```
- Создать файл ```.env```.
```shell
touch .env
```
- Заполнить файл ```.env``` по шаблону.
```python
API_TOKEN = Ваш токен
```
- Выполнить запуск
```shell
python bot.py
```
